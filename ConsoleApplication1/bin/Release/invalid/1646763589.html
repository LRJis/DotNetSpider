成员访问运算符： .->

  • 项目
  • 07/19/2022

本文内容

语法

postfix-expression:
  postfix-expression.templateoptid-expression
  postfix-expression->templateoptid-expression

注解

成员访问运算符.，用于->引用成员和structunionclass类型的成员。 成员访问表达式具有选定成员的值和类型。

有两种形式的成员访问表达式：

  1. 在第一个窗体中， postfix-expression 表示值 struct、 class类型 union 以及 id-expression 指定 struct成员的名称， union或 class。 操作的值是 l 值， id-expression 如果 postfix-expression 为 l-value，则为 l 值。

  2. 在第二个窗体中，postfix-expression表示指向classid-expressionstructunion指定成员或或名称的structunionclass指针。 该值是 id-expression 其值为 l-value。 运算符 -> 取消引用指针。 e->member表达式和(*(e)).member (，其中e表示指针) 产生相同的结果 (，除非运算符->或*重载) 。

示例

以下示例演示成员访问运算符的两种形式。

// expre_Selection_Operator.cpp
// compile with: /EHsc
#include <iostream>
using namespace std;

struct Date {
   Date(int i, int j, int k) : day(i), month(j), year(k){}
   int month;
   int day;
   int year;
};

int main() {
   Date mydate(1,1,1900);
   mydate.month = 2;
   cout  << mydate.month << "/" << mydate.day
         << "/" << mydate.year << endl;

   Date *mydate2 = new Date(1,1,2000);
   mydate2->month = 2;
   cout  << mydate2->month << "/" << mydate2->day
         << "/" << mydate2->year << endl;
   delete mydate2;
}

2/1/2000
