按位包含 OR 运算符： |

  • 项目
  • 07/19/2022

本文内容

语法

expression1|expression2

注解

按位包含 OR 运算符 (|) 将其第一个操作数的每个位与其第二个操作数的对应位进行比较。 如果其中一个位是 1，则将对应的结果位设置为 1。 否则，将对应的结果位设置为 0。

运算符的两个操作数必须具有整型类型。 标准转换中涵盖的常用算术转换将应用于操作数。

运算符关键字 |

C++ 指定 bitor 为 . 的 |替代拼写。 在 C 中，备用拼写作为 iso646.h> 标头中的<宏提供。 在 C++ 中，替代拼写是关键字; <已弃用 iso646.h> 或 C++ 等效 <ciso646> 。 在 Microsoft C++ 中 /permissive- ，启用备用拼写需要编译器 /Za 选项。

示例

// expre_Bitwise_Inclusive_OR_Operator.cpp
// compile with: /EHsc
// Demonstrate bitwise inclusive OR
#include <iostream>
using namespace std;

int main() {
   unsigned short a = 0x5555;      // pattern 0101 ...
   unsigned short b = 0xAAAA;      // pattern 1010 ...

   cout  << hex << ( a | b ) << endl;   // prints "ffff" pattern 1111 ...
}
